# AI Chatbot — General Purpose Assistant

A conversational AI assistant built with **Streamlit** and **Groq’s Llama 3.3 model**. It remembers the conversation, responds clearly, and works in real time.

## Features
- Chat memory (keeps context)
- Clean, chat‑style UI
- Powered by Llama 3.3 via Groq API
- Deployed live on Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Groq API
- Llama 3.3

## Live Demo
👉 **[Try the app here](https://ai-chatbot-ax9uzbqdejsz3bjumbsth2.streamlit.app/)**

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
